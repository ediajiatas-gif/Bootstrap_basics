Bootstrap Responsive Layout Assignment

This project is a simple responsive webpage built using Bootstrap 5. The goal of this assignment was to practice working with Bootstrap’s grid system, form components, tables, utilities, and responsive design features.

What’s Included
Registration Form
First Name and Last Name fields placed side-by-side using Bootstrap grid
Email and Password fields with required validation styles
Checkbox for agreeing to terms
A green Submit button using btn-success
Data Table
A Bootstrap-styled table with sample user data
Striped rows and hover effects for better readability
Fully responsive using table-responsive
Images & Buttons
A full-width responsive image using img-fluid
A circular profile-style image using rounded-circle
Two buttons:
One always visible on all screen sizes
One hidden on small screens using responsive display utilities
Navigation Bar
Responsive navbar with links to Home, About, and Contact
Collapses into a hamburger menu on smaller screens
Layout & Responsiveness
Built using Bootstrap containers and grid system
Fully responsive across mobile, tablet, and desktop screens
Purpose

The purpose of this project was to get comfortable using Bootstrap components and utilities to quickly build clean, responsive layouts without writing custom CSS from scratch.
